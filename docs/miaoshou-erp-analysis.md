## 一、产品采集功能

### 1.1 采集入口（左侧菜单）

| 功能 | 路径 | 说明 |
|------|------|------|
| 公用采集箱 | 产品 > 通用功能 > 公用采集箱 | 采集商品的中转站，支持批量管理 |
| 链接采集 | 产品 > 通用功能 > 链接采集 | 通过商品链接批量采集 |
| 导入采集 | 产品 > 通用功能 > 导入采集 | 批量导入商品链接或ID |
| AI工作台 | 产品 > 通用功能 > AI工作台 | AI辅助采集 |
| 侵权检测 | 产品 > 通用功能 > 侵权检测 | 检测商品侵权风险 |
| 货盘中心 | 产品 > 货盘中心 | 跨境热卖/淘宝/本土货盘 |
| 1688搜同款 | 产品 > 货盘中心 | 1688同款商品 |
| 1688智能采集 | 产品 > 货盘中心 | AI智能1688采集 |
| 妙手精选货盘 | 产品 > 货盘中心 | 平台精选货盘 |
| AI选品 | 产品 > 货盘中心 | AI智能选品 |
| 1688选品库 | 产品 > 货盘中心 | 1688货源选品库 |
| 数据选品 | 产品 > 数据选品 | 基于数据的选品 |
| 快速上货 | 产品 > 快速上货 | 快速发布商品到平台 |

**跨境货盘：**
- 美区货盘

百货/综合类、男装/女装/内衣、箱包、男鞋/女鞋、床上用品、图书、汽车用品、酒类、玩具、美妆

### 1.3 采集配置选项

- **采集并自动认领**：采集商品时自动认领到指定平台
- **采集并自动发布**：采集后直接发布到店铺
- **自动认领(发布)到平台**：可选择一个或多个平台

## 二、产品认领功能

### 2.1 认领入口
- **公用采集箱**：已采集商品在此认领
- **快速上货**：批量认领到Shopee等平台

### 2.2 支持销售平台
- Shopee
- TikTok
- Temu
- Lazada
- Amazon
- 等其他跨境平台

### 2.3 认领状态
- 未认领
- 已认领
- 采集失败

## 三、产品编辑功能

- 分组管理
- 认领平台管理
