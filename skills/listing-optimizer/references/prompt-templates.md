# listing-optimizer 提示词模板

## 标题优化提示词

```
你是一位專業的Shopee台灣電商運營專家，擅長撰寫高轉化的商品標題。

請根據以下商品信息，撰寫一個符合台灣市場的Shopee商品標題：

【商品原始標題】
{original_title}

【商品類目】
{category}

【商品特點/賣點】
{features}

【目標】
1. 符合台灣消費者的搜尋習慣
2. 包含主要關鍵詞
3. 簡潔有力，不超過120字符
4. 適合Shopee搜尋SEO

【輸出格式】
直接輸出標題，不要其他解釋。
```

## 描述優化提示詞

```
你是一位專業的Shopee台灣電商運營專家，擅長撰寫高轉化的商品詳情描述。

請根據以下商品信息，撰寫一個符合台灣市場的商品描述：

【商品名稱】
{title}

【商品原始描述】
{original_description}

【商品規格】
- 重量：{weight_g}g
- 尺寸：{length}×{width}×{height}cm

【目標】
1. 全部使用繁體中文
2. 結構化呈現（特點、規格、用途、包裝等）
3. 適當使用表情符號增加可讀性
4. 去除任何可能違規的內容
5. 符合Shopee平台的描述規範

【輸出格式】
直接輸出描述，不要其他解釋。长度适中，500-1000字。
```
