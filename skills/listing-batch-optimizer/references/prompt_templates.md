# 优化提示词模板

## 批量优化提示词

### v1: 标准七段式（保持一致性）

```
## 商品基本信息
- 商品名稱：{title}
- 標題關鍵詞：{title_keywords}
- SKU選項：共{sku_count}個規格

## 寫作要求
1. 使用台灣繁體中文
2. 嚴格遵守七段式結構：開場標題句→痛點切入→商品特點→材質與工藝→使用場景→選購參考→溫馨提示
3. 總字數：1800-2500字符
4. **重要**：描述開頭（前200字）必須包含標題中的核心關鍵詞：{title_keywords}
5. 禁止出現：現貨、熱銷、限量、免運、庫存、當天寄出
6. 不要使用"賣點"作為欄目名，改用"商品特點"

請直接輸出優化後的描述，不要解釋。
```

### v2: 强调关键词密度

```
## 商品基本信息
- 商品名稱：{title}
- 標題關鍵詞（必須在描述前200字全部出現）：{title_keywords}
- SKU數量：{sku_count}

## 關鍵詞密度要求
- {title_keywords} 這幾個詞都必須在描述前200字出現
- 每個關鍵詞至少出現1次
- 可以拆分出現（如"露營"和"燈架"分開）

## 寫作要求
1. 台灣繁體中文，1800-2500字符
2. 七段式結構
3. 禁止詞：現貨、熱銷、限量、免運、庫存、當天寄出
4. 欄目名用"商品特點"而非"賣點"

直接輸出描述。
```

### v3: 带一致性自检

```
## 商品基本信息
- 商品名稱：{title}
- 標題關鍵詞：{title_keywords}

## 自檢清單（輸出前必須確認）
☐ 描述前200字包含所有關鍵詞：{title_keywords}
☐ 總字數1800-2500
☐ 無禁用詞
☐ 七段式結構完整

## 寫作要求
1. 台灣繁體中文
2. 七段式：開場標題句→痛點切入→商品特點→材質與工藝→使用場景→選購參考→溫馨提示
3. 描述前200字必須出現：{title_keywords}
4. 禁止：現貨、熱銷、限量、免運
5. 欄目名："商品特點"

直接輸出描述，輸出前先自我檢查。
```

## 快速参考

| 项目 | 要求 |
|------|------|
| 字数 | 1800-2500字符 |
| 开头 | 前200字必须包含标题关键词 |
| 结构 | 七段式 |
| 禁用词 | 現貨、熱銷、限量、免運、庫存、當天寄出 |
| 栏目名 | 商品特點（不用賣點） |

## 维护补充

### 为什么文档要求和最终输出都强调“前200字关键词”

1. 这是当前脚本校验的硬规则，不满足就不会写回结果
2. 模型提示词已经要求前200字覆盖关键词，但真实生成仍可能偶发漏词
3. 因此脚本在模型输出后还会执行一次“关键词头部补强”，把缺失关键词强制补到描述最前面

### 什么时候该改提示词，什么时候该改补强逻辑

1. 若同类商品普遍漏掉同一类关键词，优先优化提示词
2. 若只是偶发漏词，优先保留现有提示词，依赖脚本补强兜底
3. 若补强后的描述仍不自然，再调整补强文案模板，而不是直接取消校验
