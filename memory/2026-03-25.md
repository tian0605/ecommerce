### miaoshou-updater 浏览器自动化最终调试成功

**商品1026175430866全字段填写并保存发布**

#### 关键发现

1. **类目组件类型**：`el-cascader`（级联选择器）而非 `el-dropdown`
   - cascader节点在DOM中存在但Playwright locator检测为不可见
   - 解决：用 `dialog.evaluate()` + JS `.click()` 直接点击节点

2. **Form Item 索引（编辑对话框）**：
   - [0] = 产品标题
   - [1] = 简易描述
   - [3] = 主货号
   - [5] = 类目（cascader）
   - [12] = 包裹重量
   - [13] = 包裹尺寸

3. **重量单位**：数据库存克(g)，ERP对话框填KG → 公式：`dialog_value = package_weight_g / 1000`
   - 示例：2500g → 填 2.5

4. **类目三级级联选择路径**：
   - 家居生活 (Home & Living) → 居家收纳 (Home Organizers) → 收纳盒、收纳包与篮子 (Storage Boxes, Bags & Baskets)
   - 每级需等0.3-0.5秒让面板更新

5. **标题填写**：`page.locator('input[placeholder="标题不能为空"]').fill(text)`
   - 描述填写：`page.locator('.el-dialog__body .el-form-item:nth-child(2) textarea').fill(text)`

6. **最大教训**：每次脚本都刷新页面导致数据丢失
   - 解决：所有操作在一个浏览器会话内完成，不刷新页面
