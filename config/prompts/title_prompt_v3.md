## 商品名称结构公式

1. **热搜关键词**（必须）：
   - 位于标题最前面
   - 包含1-2个热搜词

2. **核心卖点**（可选）：
   - 突出产品最大优势
   - 用2-3个精炼词汇表达
   - 如：折疊、防潮、靜音、加厚

3. **适用场景**（可选）：
   - 明确使用场景或适用人群
   - 如：床底、客廳、小坪數、租屋族

4. **产品属性**（必须）：
   - 尺寸、颜色、材质、规格
   - 如：120cm、灰色、布藝、3層

5. **促销标签**（可选）：
   - 免運、現貨、熱銷、限量
   - 位于标题最后

## 写作规范

### 长度要求
- **总字数**：40-55字符（最佳区间）
- **下限**：30字符（避免信息不足）
- **上限**：60字符（平台限制）

### 语言要求
1. **使用台湾繁体中文**
2. **台湾本地化词汇**：

```
✅ 使用：收納、折疊、床底、客廳、免運、現貨、熱銷
❌ 禁用：收纳、折叠、床底、客厅、包邮、现货、热销
```

### SEO优化原则

#### 原则1：关键词优先级排序
```
优先级排序：
热搜词（核心品类） > 核心品类词 > 卖点 > 属性
热搜词（核心品类） > 卖点 > 核心品类词
热搜词（核心品类） > 属性搜索词 > 核心品类词
```

### 违禁词与敏感词

#### 绝对禁止（广告法违规 + 跨境合规）：
```
❌ "最好"、"第一"、"顶级"、"极致"、"最"、"绝对"
❌ "永久"、"100%"、"全部"、"完全"
❌ "世界第一"、"全国第一"
❌ 夸大宣传、虚假承诺
❌ 绝对禁止出现"现货"、"現貨"、"库存充足"、"随时发货"等暗示本地库存的词汇
   （跨境界售卖的商品不能暗示本地有现货，会涉及海关/保税区合规问题）
```

#### 相对禁止（平台违规）：
```
❌ 大陆特有词汇：快递、包邮、秒杀、拼团
❌ 跨平台品牌词：其他平台品牌名
❌ 联系方式：电话、微信、QQ
❌ 引导站外交易
❌ 暗示本地发货的词汇：當天寄出、24小時出貨、快速出貨等
```

#### 建议禁用（影响用户体验）：
```
❌ 过多特殊符号：【】()（建议使用不超过2个）
❌ 过长词汇：超过10个字符的词汇
❌ 重复词汇：同一个词出现2次以上
❌ 不相关关键词：与商品无关的流量词
```

### 品牌词处理规则

#### 如果有品牌：
1. **品牌知名** → 放在标题最前面
   ```
   示例：【三洋】收納盒 折疊 防潮 3層 現貨
   ```

2. **品牌一般** → 放在核心关键词后
   ```
   示例：收納盒 折疊 【Foshan Premium】 防潮 3層
   ```

#### 如果无品牌：
- 不要虚构品牌名
- 直接以核心关键词开头

### 属性信息提取规则

#### 从中提取SKU特征：
```
1. 规格差异（尺寸、颜色、材质）
2. 找出最具代表性的SKU
3. 如果有多个SKU，用"/"分隔
   示例：30cm/40cm/50cm
```

## 常见问题处理

### 问题1：如何处理过长的1688标题？
**解决方案**：
1. 提取核心关键词（品牌+品类+核心卖点）
2. 删除形容词和修饰语（优质、精美、时尚）
3. 保留数字和规格（30cm、3層、2件）
4. 控制在40-55字符

示例：
```
原标题：优质时尚简约现代风格多功能可折叠床底鞋收纳盒
优化后：收納盒 鞋盒 折疊 多功能 現貨
```

### 问题2：SKU信息过多如何处理？
**解决方案**：
1. 选择最主流的SKU（销量最高的）
2. 如果SKU差异大，用"/"分隔
3. 不要在标题中列出所有SKU
4. 详情页中再展示完整SKU列表

示例：
```
SKU：30cm/40cm/50cm 多色可选
标题：收納盒 30cm/40cm/50cm 多色 現貨
```

### 问题3：热搜词与商品不匹配怎么办？
**解决方案**：
1. 优先选择与商品相关的热搜词
2. 不相关的热搜词不要强行融入
3. 宁缺毋滥，保证标题相关性

示例：
```
热搜词：收納盒、鞋盒、床底收納、化妝品收納
商品：鞋盒
选择：收納盒、鞋盒、床底收納（舍弃化妝品收納）
```
