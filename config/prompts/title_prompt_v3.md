## 语言硬约束（第一优先级）

### 输出语言判断规则
根据 `{listing_language}` 参数严格匹配输出语言：

| listing_language | 输出要求 |
|-----------------|---------|
| zh-Hant | 台湾繁体中文，使用本地化词汇（收納、折疊、客廳、化妝品、瀝水、廚房） |
| en | 自然英文，适合菲律宾/新加坡/马来西亚站点，禁止中文字符（除产品型号/数字外） |
| id | 印尼语，使用日常用语（Rak, Penyimpanan, Lipat, Anti Air） |
| th | 泰语，使用泰文字符和本地表达 |
| vi | 越南语，保留声调符号，使用本地化词汇 |
| ms | 马来语，使用标准马来文 |
| pt | 巴西葡萄牙语（非欧洲葡语） |
| es | 墨西哥西班牙语 |

### 语言转换要求
- 中文源 → 非中文站点：必须完整翻译，禁止中英混杂
- 中文源 → 中文站点：繁简转换 + 台湾本地化词汇替换
- 除产品型号、容量数字、尺寸数字外，**禁止保留源语言字符**

## 标题结构公式

```
[热搜关键词] + [核心卖点] + [适用场景] + [规格属性]
```

**位置优先级：**
1. **前段（0-20字符）**：热搜关键词 + 核心品类词
2. **中段**：核心卖点（2-3个）+ 使用场景
3. **后段**：规格属性（尺寸/颜色/材质）+ 变体差异

**多SKU处理：**
- 存在多个核心SKU差异时，用 `/` 合并2-3个代表性变体
- 示例：`30cm/40cm/50cm` 或 `White/Black/Grey`

## 长度要求

| 站点语言 | 建议长度 | 最大限制 | 说明 |
|---------|---------|---------|------|
| zh-Hant | 40-55字 | 60字 | 中文字符计数 |
| en | 45-90 chars | 100 chars | 英文字符计数 |
| id/vi/ms | 50-100 chars | 120 chars | 拉丁字母类 |
| th | 40-70字 | 80字 | 泰文字符计数 |
| pt/es | 50-100 chars | 120 chars | 拉丁字母类 |

**原则：** 信息充分但不过长，避免截断影响展示。

## SEO优化原则

### 关键词优先级排序
```
热搜词（核心品类）> 核心品类词 > 卖点词 > 属性词
```

### 融入策略
✅ 优先融入1-2个最相关热搜关键词
✅ 关键词自然分布，不堆砌
✅ 保留原商品真实品类，不偏题
✅ 关键属性（颜色/材质/容量/尺寸/场景）必须体现

❌ 不相关热搜词不要强行融入
❌ 同一词不要重复超过2次
❌ 不要为了凑长度添加无意义词汇

## 跨境合规禁止事项

### 绝对禁止（所有站点）
```
❌ 本地库存暗示：
   现货/現貨/Ready Stock/库存充足/当天寄出/24小时出货
   
❌ 物流优惠承诺：
   包邮/免运/Free Shipping/运费优惠
   
❌ 强刺激营销：
   热销/爆款/Best Seller/限量/Limited/立即购买/Buy Now
   
❌ 夸大宣传：
   最好/第一/顶级/极致/绝对/永久/100%/世界第一
   
❌ 站外引导：
   其他平台品牌名、联系方式（电话/微信/Line/WhatsApp）
```

### 站点特定禁止
- **台湾站**：禁止简体字、大陆网络用语
- **东南亚站**：禁止过于正式的书面语，使用日常用语
- **拉美站**：禁止使用欧洲葡语/西语表达

## 输出要求

**只输出一个最终标题。**

❌ 不要输出解释、分析过程、多个候选标题
❌ 不要输出引号、项目符号、编号
❌ 不要输出英文翻译对照（非英文站点）

**输出格式：** 直接输出标题文本，无任何额外内容。
