# Shopee 全站点商品描述生成提示词 v1.0

## 运行上下文
- 站点：{site_code}
- 语言：{listing_language}
- 商品名称：{product_name}
- 材质：{material}
- 核心特点：{features}
- 适用场景：{scenarios}
- 热搜关键词：{hot_search_words}

## 任务目标
生成适合目标站点搜索与转化的商品描述，内容必须符合目标站点语言、阅读习惯与跨境合规要求。

---

## 语言硬约束（第一优先级）

### 输出语言判断规则
根据 `{listing_language}` 参数严格匹配输出语言：

| listing_language | 输出要求 |
|-----------------|---------|
| zh-Hant | 台湾繁体中文，使用本地化词汇与表达 |
| en | 自然英文，适合菲律宾/新加坡/马来西亚站点，禁止中文字符（除型号/数字外） |
| id | 印尼语，使用印尼日常电商表达 |
| th | 泰语，使用泰文本地表达 |
| vi | 越南语，保留声调符号，使用本地化词汇 |
| ms | 马来语，使用标准马来文 |
| pt | 巴西葡萄牙语（非欧洲葡语） |
| es | 墨西哥西班牙语 |

### 语言转换要求
- 中文源 → 非中文站点：必须完整翻译，禁止中英混杂
- 中文源 → 中文站点：繁简转换 + 台湾本地化词汇替换
- 除产品型号、容量数字、尺寸数字外，**禁止保留源语言字符**

---

## 描述结构要求

请按适合目标站点的自然语言生成结构化描述，至少覆盖以下内容：

1. 开场引入：直接说明产品用途、适用痛点或使用价值
2. 商品特点：整理 4-6 条核心卖点，突出容量、结构、材质、使用便利性
3. 材质与规格：说明材质、结构、主要规格或变体差异
4. 适用场景：列出 3-4 个使用场景、人群或摆放位置
5. 使用建议：给出 2-3 条选购建议、收纳建议或使用提示
6. 注意事项：补充 2-3 条温馨提示、清洁方式或尺寸提醒

如目标语言更适合短段落或小标题，可用自然的小标题组织内容，但不要写成模板化空话。

---

## 关键词与 SEO 要求

1. 自然融入 `{hot_search_words}` 中最相关的关键词
2. 前 200 字必须自然包含标题关键词和核心品类词
3. 不要堆砌关键词，不要连续重复同一词汇
4. 关键属性必须体现：材质、颜色、尺寸、容量、适用场景、结构特点

---

## 风格要求

✅ 像目标站点本地卖家一样写，清晰、可信、易读
✅ 使用分段、列表或自然小标题增强可读性
✅ 保留原商品真实品类，不偏题
✅ 强调真实卖点，不做机械直译

❌ 不要写成硬广，不要强迫下单
❌ 不要整段堆砌规格词和关键词
❌ 不要照抄原始标题或生成空洞套话

---

## 字数要求

| 站点语言 | 建议长度 | 最大限制 |
|---------|---------|---------|
| zh-Hant | 1200-2500 字符 | 3000 字符 |
| en | 900-1800 chars | 2200 chars |
| id/vi/ms | 900-1800 chars | 2200 chars |
| th | 800-1600 字符 | 2000 字符 |
| pt/es | 900-1800 chars | 2200 chars |

---

## 跨境合规禁止事项

### 绝对禁止（所有站点）
```
❌ 本地库存暗示：
	现货/現貨/Ready Stock/库存充足/当天寄出/24小时出货
   
❌ 物流优惠承诺：
	包邮/免运/Free Shipping/运费优惠
   
❌ 强刺激营销：
	热销/爆款/Best Seller/限量/Limited/立即购买/Buy Now
   
❌ 夸大宣传：
	最好/第一/顶级/极致/绝对/永久/100%/世界第一
   
❌ 站外引导：
	其他平台品牌名、联系方式（电话/微信/Line/WhatsApp）
```

### 站点特定禁止
- **台湾站**：禁止简体字、大陆网络用语
- **东南亚站**：禁止过于正式的书面语，使用日常用语
- **拉美站**：禁止使用欧洲葡语/西语表达

---

## 输出要求

**只输出一个最终商品描述。**

❌ 不要输出解释、分析过程、多个候选
❌ 不要输出 JSON、代码块、引号或编号
❌ 不要输出双语对照

**输出格式：** 直接输出描述正文，无任何额外说明。
